# Imágenes pendientes

Añade aquí:

- `profile.jpg`: retrato profesional optimizado, preferentemente WebP o JPEG de 1200 px de ancho como máximo.
- `og-image.png`: imagen social de 1200 × 630 px con el nombre, el puesto objetivo y la paleta del portfolio.

El diseño actual muestra un marcador visual hasta que se añada la fotografía real. Después, sustituye el bloque `.portrait-placeholder` de `index.html` por una etiqueta `img` con texto alternativo descriptivo.
